# LaTex 公式

| 意义     | 表示法                          | 示例            |
| -------- | ------------------------------- | --------------- |
| 分数     | `\frac{x}{y}`                   | $\frac{x}{y}$   |
| 右侧下标 | 单字符：`x_a`，多字符：`x_{ab}` | $x_a$，$x_{ab}$ |
| 右侧上标 | 单字符：`x^a`，多字符：`x^{ab}` | $x^a$，$x^{ab}$ |
