# 逻辑运算

## 与

同时为 1 结果才为 1。

表示：$A \&\& B$，$A \land B$。

```
0 && 0 = 0
1 && 0 = 0
0 && 1 = 0
1 && 1 = 1
```

## 或

任一为 1，结果就为 1。

表示：$A || B$ ，$A \lor B$，$A + B$。

```
0 || 0 = 0
1 || 0 = 1
0 || 1 = 1
1 || 1 = 1
```

## 非

表示：$!A$，$\neg A$, $\overline{A}$。

```
!0 = 1
!1 = 0
```

## 异或

相同为 0，不同为 1。

表示：$A \operatorname{XOR} B$，$A \oplus B$。

```
a XOR b = ( !a && b ) || ( a && !b )
```

```
0 XOR 0 = 0
1 XOR 0 = 1
0 XOR 1 = 1
1 XOR 1 = 0
```

## 与非

任一为 0，结果为 1。

表示：$A \operatorname{NAND} B $。

```
a NAND b = !( a && b )
```

```
0 NAND 0 = 1
1 NAND 0 = 1
0 NAND 1 = 1
1 NAND 1 = 0
```
